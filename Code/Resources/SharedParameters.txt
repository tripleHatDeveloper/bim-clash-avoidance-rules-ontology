# Clash Avoidance Plugin — Shared Parameters File
# =====================================================================
# HOW TO USE:
#   1. In Revit: Manage → Shared Parameters → Browse to this file
#   2. Manage → Project Parameters → Add
#   3. Select the parameter group "Clash Avoidance" and the parameter
#      you want to bind, then assign it to the relevant category.
#
# PARAMETER BINDING RECOMMENDATIONS:
#
#   CA_CeilingHeightThreshold  → Bind to: Walls (instance)
#                                OR Project Information (project-wide default)
#                                Value: ceiling height in mm, e.g. 2700
#                                This sets the maximum ceiling line height.
#
#   CA_MEPClearance            → Bind to: Walls (instance)
#                                OR Project Information (project-wide default)
#                                Value: MEP zone clearance in mm, e.g. 300
#                                Rule A02 checks: WallHeight ≤ Ceiling − MEP
#
#   CA_DimensionalThreshold    → Bind to: any element category where a
#                                dimensional limit has been agreed in the BIM
#                                Execution Plan (e.g. structural framing,
#                                mechanical equipment).
#                                Value: maximum allowed dimension in mm.
#
# NOTE: All values are entered in MILLIMETRES in the Revit UI.
#       The plugin converts mm → decimal feet internally.
# =====================================================================
